# Documentação (padrão do projeto)

- **Tudo novo** deve ser criado em `docs/` (arquivos .md, .pdf, etc.)
- **Histórico / cronologia** fica em `docs/cronologia/`
  - Criar uma pasta por data: `docs/cronologia/YYYY-MM-DD/`
  - **Nunca apagar** entradas antigas (sempre acrescentar).
- A antiga pasta `reports/` (raiz) não deve ser versionada. Se precisar gerar relatórios locais,
  pode usar `reports/`, mas ela ficará ignorada pelo Git.
